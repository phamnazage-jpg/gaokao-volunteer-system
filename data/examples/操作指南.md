## 🔧 第一步：配置 Gmail

### 1.1 获取 Gmail 应用专用密码

1. 登录 Google 账号：https://myaccount.google.com
2. 点击 **安全性**
3. 开启 **两步验证**（如未开启）
4. 搜索 **应用专用密码** 或进入：https://myaccount.google.com/apppasswords
5. 选择应用：**邮件**
6. 选择设备：**其他（自定义名称）**
7. 输入名称：**Himalaya CLI**
8. 点击 **生成**
9. **记录16位应用专用密码**（类似：`abcd efgh ijkl mnop`）

> ⚠️ **重要**：不是Gmail登录密码，是专门的**应用专用密码**

### 1.2 设置环境变量

在终端中运行：

```bash
export GMAIL_APP_PASSWORD="abcd...n添加到 `.bashrc` 使其永久生效：

```bash
echo 'export GMAIL_APP_PASSWORD="abcd...9"' >> ~/.bashrc
source ~/.bashrc
```

## 🆘 常见问题

### Q1: "GMAIL_APP_PASSWORD is not set"

**解决**：先设置环境变量
```bash
export GMAIL_APP_PASSWORD="你的应用...n### Q2: 邮件发送失败，提示认证错误

**解决**：
1. 确认使用的不是Gmail登录密码，而是**应用专用密码**
2. 重新生成应用专用密码（Google账号 → 安全性 → 应用专用密码）
3. 检查密码是否正确设置到环境变量

### Q3: "Please log in via your web browser"

**解决**：Gmail 认为登录可疑
1. 登录 https://mail.google.com
2. 查看是否有安全警告邮件
3. 访问 https://myaccount.google.com/lesssecureapps
4. 或访问 https://accounts.google.com/DisplayUnlockCaptcha 解除锁定
