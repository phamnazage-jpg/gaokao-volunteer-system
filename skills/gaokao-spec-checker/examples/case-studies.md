# 常见错误案例库

## 案例1：原始方案（错误版）

### 错误内容

```
湖南578分考生志愿方案

本次共填报45个学校志愿：
志愿01：江西财经大学，会计学
志愿02：长沙理工大学，财务管理
...

录取概率：
志愿01 35%
志愿02 45%
...

投档后如果不服从调剂，会被退到下个志愿。

所有会计学专业都要求物理+化学+生物选科。
```

### 错误清单

| 错误                 | 严重性  | 修正                     |
| -------------------- | ------- | ------------------------ |
| 志愿单位是"45个学校" | 🔴 致命 | 改为"45个院校专业组"     |
| 退到下个志愿         | 🔴 致命 | 改为"进入征集志愿"       |
| 主观概率             | 🟡 严重 | 删除，改为2025年参考位次 |
| 选科一刀切           | 🟡 严重 | 逐校核实                 |

## 案例2：修正版（仍可优化）

### 改进点

- ✅ 正确使用"院校专业组"
- ✅ 提到"组内专业服从"
- ✅ 2025年参考位次

## 复盘教训

### 我自己犯过的错误

1. **第一版方案**：用"45个学校"为单位 → 致命错误
2. **第一版方案**：说"退到下个志愿" → 致命错误
3. **第一版方案**：给"35%、80%"主观概率 → 严重错误
4. **第一版方案**：所有会计都说"物+化+生" → 严重错误

### 错误根源

1. **政策更新滞后**：还在用2024年的旧概念
2. **数据主观化**：没有基于真实位次
3. **批量假设**：对所有学校用同一选科要求
4. **风险意识弱**：没有充分提示退档风险

### 改进方向

- ✅ 每次方案生成后**立即调用检查器**
- ✅ 关键政策每年更新时**重新学习**
- ✅ **逐校核实**而不是批量假设
- ✅ 风险提示必须**前置**而不是省略
