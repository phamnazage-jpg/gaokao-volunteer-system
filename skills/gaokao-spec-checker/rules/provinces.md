## 快速识别规则

### 识别关键词

- "院校专业组" → 该模式
- "专业（类）+学校" → 浙江/山东模式
- "院校志愿" → 传统模式

### 数字范围

- 80-112个 → 大概率专业+学校模式
- 30-50个 → 大概率院校专业组模式
- 1-10个 → 大概率传统模式
