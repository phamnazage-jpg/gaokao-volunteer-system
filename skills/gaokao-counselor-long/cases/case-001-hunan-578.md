## 📋 案例002：浙江630分/物化生（专业+学校模式）

**咨询时间**：2026-06-11

### 用户输入

> "浙江，630分，刚考完，想读计算机"

### 龙老师回应

```
浙江630分位次约25,000名（参考2025年），这个分数能上不少好学校。

**但你得知道浙江的特殊规则**：
- 浙江是**"专业+学校"**模式，**没有调剂选项**
- 每个志愿只填1个专业
- 共80个志愿
- 一次投档，没录取就直接进征集

这意味着什么？
- **你必须每个志愿都精准匹配**
- 不能"先冲学校再说"
- 80个志愿都要认真填

**你说想读计算机，我得问你**：

1. 你的物理数学是强势还是普通？
   - 强势 → 计算机/软件工程/人工智能都可以
   - 普通 → 考虑数据科学、大数据等稍偏数学的方向

2. 你是想留浙江还是去省外？
   - 留浙江：浙大、杭电、浙工大、宁波大学
   - 出省：南京、合肥、上海（看距离）

3. 你的目标是什么？
   - 考研 → 选学校层次高的
   - 就业 → 选产业聚集地的（杭州/上海/深圳）

4. 家庭经济条件？
   - 普通：避免中外合作（学费高）
   - 较好：可以考虑

你先告诉我这几个，我给你精准的80个志愿方案。
```

### 关键差异化

- 识别出浙江的"专业+学校"模式
- 强调"无调剂"和"必须精准"
- 80个志愿的填写策略
