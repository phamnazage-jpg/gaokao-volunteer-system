# 可视化报告系统

## 为什么需要可视化

**传统报告问题**：
- ❌ 数据堆叠，难以抓住重点
- ❌ 对比关系不清晰
- ❌ 缺乏直观风险感知

**可视化优势**：
- ✅ 雷达图 → 四维度匹配度一眼可见
- ✅ 热力图 → 专业匹配度颜色编码
- ✅ 对比表 → 院校优劣直观展现
- ✅ 红绿灯 → 高风险项立即警示

---

## 可视化元素

### 1. 考生画像雷达图

四维度评分（1-10分）：
- **兴趣匹配度**: 霍兰德测试结果
- **能力匹配度**: 学科成绩匹配程度
- **就业匹配度**: 专业就业前景
- **家庭适配度**: 经济条件、地域要求

**综合得分公式**:
```
综合得分 = 兴趣×30% + 能力×30% + 就业×25% + 家庭×15%
```

| 等级 | 图标 | 说明 |

## 使用方法

### 生成可视化报告

```bash
# 方式1：命令行直接运行
python3 ~/.local/bin/gaokao-visual-report-v2.py

# 方式2：Python导入
from gaokao_visual_report_v2 import generate_visual_report

files = generate_visual_report(student_data, volunteer_list, output_format='all')
# 返回: [md_file, html_file, pdf_file]
```

### 输出格式

| 格式 | 大小 | 用途 |
|------|------|------|
| **Markdown** | ~8KB | 可编辑文档 |
| **HTML** | ~14KB | 网页展示、交互图表 |
| **PDF** | ~340KB | 打印、分享、存档 |
