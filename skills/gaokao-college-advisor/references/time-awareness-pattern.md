# 高考志愿填报时间感知模式

## 为什么时间感知很重要

高考志愿填报是一个时间敏感的任务。不同时间点的咨询场景完全不同：
- 6月7-9日：高考进行中
- 6月10-24日：高考结束，等待出分
- 6月25日左右：成绩公布，开始填报
- 6月底：填报 deadline
- 7月：录取结果公布
- 8月：补录/征集志愿
- 9月后：为下一年做准备

---

## 时间感知模式

### 模式1：当前年份判断

```python
def get_current_year():
    """获取当前年份（高考年份）"""
    from datetime import datetime
    current_date = datetime.now()
    
    # 如果当前日期在1月-6月，可能是当年高考
    # 如果当前日期在7月-12月，主要是为下一年做准备
    if current_date.month <= 6:
        return current_date.year
    else:
        return current_date.year + 1
```

### 模式2：高考阶段判断

```python
def get_gaokao_stage(current_date):
    """
    判断当前处于高考哪个阶段
    
    返回：
    - 'before_exam': 高考前（1月-6月6日）
    - 'during_exam': 高考进行中（6月7-9日）
    - 'after_exam': 高考结束到出分（6月10-24日）
    - 'score_release': 成绩公布（6月25日左右）
    - 'filling': 志愿填报期（6月25日-7月初）
    - 'admission': 录取阶段（7月）
    - 'supplement': 补录/征集志愿（7-8月）
    - 'next_year_prep': 下一年准备（9月-12月）
    """
    month = current_date.month
    day = current_date.day
    
    if month < 6 or (month == 6 and day < 7):
        return 'before_exam'
    elif month == 6 and 7 <= day <= 9:
        return 'during_exam'
    elif month == 6 and 10 <= day <= 24:
        return 'after_exam'
    elif month == 6 and day >= 25:
        return 'score_release'
    elif month == 7:
        if day <= 10:
            return 'filling'
        else:
            return 'admission'
    elif month == 8:
        return 'supplement'
    else:  # 9-12月
        return 'next_year_prep'
```

### 模式3：参考数据年份确定

```
当前日期决定参考数据年份：

如果在当年6月25日之后：
- 主要参考数据 = 上一年（如2026年6月25日后，参考2025年数据）
- 辅助参考数据 = 上上年（如2025年6月25日后，同时参考2024年数据）
- 当年招生计划 = 待公布（预计6月中下旬）

如果在当年6月25日之前：
- 主要参考数据 = 上一年（如2025年）
- 辅助参考数据 = 上上年（如2024年）
```

## 不同阶段的咨询策略

### 阶段1：高考前（1月-6月6日）

**用户意图**：提前规划、了解规则、模拟填报

**核心建议**：
- 强调"位次法"的重要性
- 介绍填报规则和流程
- 帮助建立初步的院校认知
- 强调：现在只是提前了解，最终方案要等出分后

**话术示例**：
```
"您提前了解是对的，但记住：现在只是打基础。
真正靠谱的方案，要等6月25日出了分数和位次才能定。
现在可以做的就是：
1. 了解填报规则
2. 划定一个大范围的可选院校
3. 等出分后再精准匹配"
```

### 阶段2：高考后出分前（6月10-24日）

**用户意图**：预估分数、提前规划

**核心建议**：
- 根据预估分数给出参考建议
- 强调预估仅供参考
- 准备多个方案（高分/中分/低分方案）
- 等出分后立即精准定位

**话术示例**：
```
"高考刚结束，您可以先根据预估分数了解一下大致范围。
但我跟您说，估分往往不准，有的学生估分能差30分。
现在可以准备几个方案：
- 如果分数比预估高20分，怎么报？
- 如果比预估低20分，怎么报？
等6月25日出了真实位次，我们再精确定稿。"
```

### 阶段3：成绩公布后（6月25日后）

**用户意图**：正式填报，需要最终方案

**核心建议**：
- 这是正式填报阶段
- 必须基于真实位次分析
- 给出冲稳保具体方案
- 强调填报时间节点

**话术示例**：
```
"分数和位次出来了，现在可以正式填报了。
您孩子是XXX分，XXX位次，在XX省。
这个位次能上什么学校？我给您分析...

⚠️ 重要：XX省份的填报截止时间是X月X日，
千万别错过！"
```

## 常见时间相关错误

### 错误1：混淆年份

**错误示例**：
> "根据2026年录取数据..."

**问题**：如果当前是2026年，但成绩还没公布，2026年数据还不存在

**正确说法**：
> "根据2025年录取数据（2026年最新数据待6月中下旬公布）..."

### 错误2：忽略填报截止时间

**错误示例**：
只给方案，不强调 deadline

**问题**：用户错过填报时间

**正确做法**：
每次咨询必须提醒：
> "XX省的填报截止时间是X月X日X点，请务必在此之前完成填报。"

### 错误3：未区分提前规划和正式填报

**错误示例**：
6月份给提前咨询的用户说：
> "根据你的分数，我建议报XXX大学"

**问题**：6月还没出分，分数不确定

**正确说法**：
> "现在还没出分，我只能根据您的预估分数给出参考建议。
> 等6月25日出分后，我们才能给出最终方案。"

## 时间感知检查清单

每次咨询前必须确认：
- [ ] 当前日期是哪一天？
- [ ] 当前处于高考哪个阶段？
- [ ] 参考数据应该使用哪一年的？
- [ ] 当年招生计划是否已经公布？
- [ ] 填报截止时间是什么时候？
- [ ] 用户是提前规划还是正式填报？
