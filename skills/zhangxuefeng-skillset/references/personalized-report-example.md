# 个性化高考志愿报告示例

## 报告结构说明

一份高质量的个性化志愿报告应该包含以下结构：

```
1. 考生基本信息
2. 个性化档案（兴趣/能力/性格分析）
3. 数据分析总结（位次对标）
4. 志愿方案（冲稳保分类，每所学校说明推荐理由）
5. 专业对比分析
6. 个性化风险提示（基于考生情况反向排除）
7. 填报时间节点
8. 填报顺序建议
9. 给家长的建议
10. 联系方式与免责声明
```

---

## 关键点：为什么推荐

每个推荐必须说清楚这三点（张雪峰式）：

### 1. 为什么是这个

> 基于你刚才说的，孩子喜欢XX（兴趣），而且XX学科比较强（能力）。
> 这个专业既能满足孩子的兴趣，就业也有保障（就业）。
> 而且你说过希望XX（职业目标），这个专业正好对口。

### 2. 潜在风险

> 但是，我跟你说，这个专业也有坑：
> 第一，XX方面需要注意；
> 第二，如果孩子XX（能力/兴趣），会比较吃力。

### 3. 替代方案

> 如果分数不够这个专业，可以考虑XX（相近专业）。
> 如果孩子学着学着发现不喜欢，可以转方向到XX。

## 张雪峰式话术模板

### 信息收集（必须问）

```markdown
"你别跟我说'都行'，'都行'就是坑孩子。

1. 孩子平时最喜欢干啥？（不是学习，是下课、放假后主动干的事）
   - 喜欢拆东西/装东西 → 工科方向
   - 喜欢画画/设计 → 创意方向  
   - 喜欢打游戏但喜欢研究攻略 → 技术方向
   - 喜欢组织活动/当班干部 → 管理方向
   - 喜欢帮助别人/当志愿者 → 师范/医学方向

2. 孩子明确不喜欢啥？（反向排除最重要）
   - 讨厌数学 → 别碰金融、计算机、人工智能
   - 讨厌化学 → 别碰医学、药学、材料
   - 讨厌背东西 → 别碰法学、医学、文史哲
   - 讨厌和人打交道 → 别碰营销、管理、师范

3. 家里情况是啥？
   - 需要孩子本科毕业就赚钱？→ 技术类、师范类
   - 能支持孩子读研读博？→ 基础学科、医学类
   - 有特定行业资源？→ 选能接上资源的
   - 必须回XX城市工作？→ 选该市认可度高的学校

4. 孩子性格咋样？
   - 内向安静 → 技术类、研究类
   - 外向活跃 → 管理类、营销类
   - 抗压强 → 医学、金融、计算机
   - 抗压弱 → 师范、公务员导向专业

你把这四个问题答清楚，我给的方案才能靠谱。
答不清楚，我就是在瞎蒙。"
```

## 使用建议

1. **每次生成报告前**：先完整收集考生的兴趣、能力、性格、家庭背景信息
2. **使用霍兰德模型**：判断兴趣类型，匹配合适的专业大类
3. **反向排除**：根据能力弱项和明确讨厌的事项排除不合适专业
4. **个性化推荐话术**：每个推荐都说明"为什么适合你"、"潜在风险"、"替代方案"
5. **风险提示**：特别强调该考生需要特别注意的不匹配项

## 输出格式

推荐使用颜色标记：
- 🔴 冲刺志愿
- 🟡 稳妥志愿
- 🟢 保底志愿
- ⭐ 首选推荐
- ⚠️ 风险提示
- ✅ 正向匹配
- ❌ 反向排除

这样一目了然，家长考生容易理解。
