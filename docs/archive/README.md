# 历史归档索引

本目录用于存放已经明确退出“当前真相源”链路、但仍保留追溯价值的历史快照。

使用原则：

- 当前状态、当前执行板、当前仍有效问题清单，继续保留在 `docs/` 顶层。
- 已明确标注为“历史快照”、且不再被当前文档链路直接引用的文件，优先迁入本目录。
- 迁入归档目录后，仍应保留原始日期、原始上下文和真相源跳转说明，避免历史失真。
