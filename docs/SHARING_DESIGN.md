**版本**: v1.0

## 1. 功能概述

### 1.1 需求背景

用户希望将生成的志愿方案分享给：

- 家人（父母、配偶）
- 老师/规划师
- 朋友（征求意见）
- 朋友圈/社交群组（案例展示）

### 1.2 核心场景

| 场景     | 分享对象  | 分享内容 | 权限需求       |
| -------- | --------- | -------- | -------------- |
| 家庭决策 | 爸爸妈妈  | 完整方案 | 可查看、可评论 |
| 征求意见 | 老师/朋友 | 方案摘要 | 仅查看、提建议 |
| 案例展示 | 朋友圈/群 | 脱敏版本 | 仅展示         |
| 备份存档 | 自己邮箱  | 完整报告 | 永久保存       |

## 2. 分享方式

### 2.1 分享类型

| 类型           | 格式    | 适用场景      | 技术方案        |
| -------------- | ------- | ------------- | --------------- |
| **链接分享**   | URL     | 微信/短信分享 | 短链接+访问权限 |
| **图片分享**   | PNG/JPG | 朋友圈/社交   | HTML转图片      |
| **PDF分享**    | PDF文件 | 邮箱/微信文件 | 报告附件        |
| **小程序卡片** | (不做)  | -             | 跳过            |
| **HTML嵌入**   | iframe  | 博客/网站     | 嵌入代码        |

### 2.2 链接分享（核心）

#### 设计方案

```
分享链接结构:
https://gk.example.com/s/ABC123DEF

链接特点:
- 短链接（6-10位随机字符）
- 带权限控制（只读/评论/编辑）
- 有效期可选（永久/7天/30天）
- 访问密码可选
```

#### 权限级别

| 权限     | 功能           | 适用场景             |
| -------- | -------------- | -------------------- |
| **只读** | 查看方案       | 朋友圈展示、老师查看 |
| **评论** | 查看+评论建议  | 征求家人意见         |
| **编辑** | 查看+修改      | 家庭协作编辑         |
| **完全** | 查看+编辑+删除 | 仅自己               |

### 2.3 图片分享

### 2.4 PDF分享

## 3. 技术实现

### 3.1 分享链接系统

#### 数据模型

```python
class ShareLink:
    id: str              # 短链接ID (ABC123)
    report_id: str       # 关联报告ID
    owner_id: str        # 创建者ID
    permission: str      # read/comment/edit
    password: str?       # 访问密码（可选）
    expires_at: datetime? # 过期时间（可选）
    access_count: int    # 访问统计
    created_at: datetime
```

#### 访问控制流程

```
用户点击分享链接
    ↓
检查链接是否存在
    ↓
检查是否过期
    ↓
检查是否需要密码
    ↓
记录访问日志
    ↓
根据权限显示不同界面
```

### 3.2 分享生成流程

```
用户点击"分享"按钮
    ↓
弹出分享选项
    ↓
选择分享类型（链接/图片/PDF）
    ↓
设置权限（只读/评论/编辑）
    ↓
生成分享内容
    ↓
复制到剪贴板/直接分享
```

## 4. 权限与隐私

### 4.1 隐私保护

| 隐私级别 | 脱敏内容                     | 适用场景  |
| -------- | ---------------------------- | --------- |
| **完整** | 姓名/分数/位次全部显示       | 仅家人    |
| **部分** | 姓名脱敏（张\*\*）、分数显示 | 朋友/老师 |
| **完全** | 仅显示位次段、院校建议       | 公开展示  |

### 4.2 撤销分享

```
用户随时可:
1. 查看自己分享的所有链接
2. 撤销特定链接（使其失效）
3. 修改链接权限
4. 查看链接访问统计
```
