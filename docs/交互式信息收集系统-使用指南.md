# 交互式信息收集系统 - 使用指南

## 📋 系统概述

交互式信息收集系统旨在通过**分步骤引导**的方式，系统性地收集考生的完整信息，确保志愿填报推荐的准确性和个性化。

**核心特点**：
- ✅ **7步标准化流程**：从基本信息到偏好设置
- ✅ **霍兰德兴趣测试**：科学评估职业兴趣类型
- ✅ **能力量化评估**：识别优势和短板学科
- ✅ **家庭背景考量**：兼顾资源约束
- ✅ **数据验证机制**：实时检查输入合法性

---

## 🚀 使用方法

### 方式1：Python脚本（完整交互）

```bash
python3 ~/.local/bin/gaokao-collect-info.py
```

这将启动完整的7步交互式问答流程。

### 方式2：在对话中使用（推荐）

当考生咨询时，可以直接使用以下结构化问卷：

## ✅ 质量检查

收集信息时，确保：

- [ ] 基本信息完整（姓名、省份、分数、位次）
- [ ] 兴趣测评有结果（霍兰德代码）
- [ ] 能力评估有区分（强项/弱项）
- [ ] 职业目标清晰（毕业规划）
- [ ] 家庭条件考虑（经济、地域）
- [ ] 偏好明确（院校优先/专业优先）

## 💡 快速使用提示

**在对话中收集信息时**：

1. **一次一个步骤**：不要一次性问太多问题
2. **提供示例**：给选项和示例回答
3. **实时确认**：核对关键数据（如位次）
4. **解释原因**：让用户知道为什么需要这些信息
5. **鼓励完整**：强调信息越完整推荐越精准

**示例话术**：

```
"为了给您最合适的建议，我需要了解几个方面。
我们先从基本信息开始：

1. 考生姓名？
2. 所在省份？
3. 高考总分和位次？（位次一分一段表可查询）

这些信息是精准匹配的基础，请尽量准确。"
```
