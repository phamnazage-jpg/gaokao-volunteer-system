## 🔄 后续可优化方向

### 阶段1：出分前（6月11-24日）- 数据精确化

#### 1.1 逐校精确核实
- [ ] **选科要求**：登录阳光高考网，逐一核实20所目标院校的选科要求
- [ ] **2025年位次**：查每所院校每个专业组的实际录取位次
- [ ] **2025年专业组划分**：了解每所学校分几个组
- [ ] **单科要求**：检查是否有英语/数学单科线
- [ ] **体检要求**：确认色盲、色弱等限制

#### 1.2 增加院校覆盖
- [ ] 补充遗漏的优质院校
- [ ] 增加**省内重点**院校（如中南大学、湖南大学虽可能分高，但应查）
- [ ] 增加**省外知名**财经院校（中央财大、东北财大等）
- [ ] 增加**中外合作**院校（学费高但专业好）

#### 1.3 扩充专业范围
- [ ] 不限于会计/财务，加入**经济统计学**、**金融数学**等
- [ ] 考虑**数据科学与大数据技术**（物理+数学强）
- [ ] 考虑**信息管理与信息系统**（物理+管理结合）
