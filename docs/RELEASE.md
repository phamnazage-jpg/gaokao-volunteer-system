# 版本发布说明

## v2.0 (2026-06-11)

### 🎉 新功能

#### 新增角色：龙老师

- ✨ 全新的 `gaokao-counselor-long` Skill
- ✨ 定位为"张雪峰思路 + AI技术"融合体
- ✨ 标准化咨询流程
- ✨ 自动规范检查集成

#### 多省份支持

- ✨ 支持27个省份自动识别
- ✨ 区分三种模式：院校专业组 / 专业+学校 / 传统
- ✨ 自动匹配合规规则

#### 自动化测试

- ✨ 完整的测试套件
- ✨ 6个测试用例全通过
- ✨ 覆盖省份简称识别、方案检查

#### 项目化

- ✨ 完整项目目录结构
- ✨ Git版本控制
- ✨ 完整文档体系

### 🔧 改进

#### 数据精度

- 基于2025年实际录取位次
- 明确标注数据来源
- 2026年待官方公布提示

#### 规范检查

- 致命/严重/警告三级分类
- 15种错误模式库
- 自动检测和修正建议

#### 可视化报告

- HTML / PDF / Markdown三格式
- 雷达图、热力图、对比表
- 风险检测红绿灯

### 🐛 已修复

#### 致命错误

- ❌ "45个学校" → ✅ "45个院校专业组"
- ❌ "退到下个志愿" → ✅ "进入征集志愿"
- ❌ "全校调剂" → ✅ "组内专业调剂"

#### 严重错误

- ❌ 主观概率 → ✅ 位次参考
- ❌ 选科一刀切 → ✅ 逐校核实
- ❌ 数据年份混乱 → ✅ 明确2025年

## 历史版本

### v1.0 (2026-06-10 → 2026-06-11)

#### 初始版本

- gaokao-college-advisor Skill
- 可视化报告脚本
- 1分钟/3分钟/7步三套问卷
