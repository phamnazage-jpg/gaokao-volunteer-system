# 服务说明与使用条款（草案）

最后更新: 2026-06-14  
适用范围: 高考志愿填报智能系统 T12 用户端 Web 自助 MVP

## 1. 服务性质

本服务提供高考志愿填报相关的信息整理、规则检查、风险提示、审核建议和报告生成能力。  
本服务属于**辅助决策工具与咨询支持服务**，不构成对录取结果的保证。

## 2. 用户提交责任

用户或监护人提交资料时，应保证：

- 分数、位次、选科等信息真实
- 有权提交相关个人资料
- 已获得监护人知情同意（如适用）

## 3. 系统能力边界

我们会尽力提供：

- 套餐下单与订单状态展示
- 规则校验与风险提示
- 报告查看与 PDF 下载
- 售后、退款与补充资料的最小兜底

但不承诺：

- 被任何学校或专业录取
- 所有数据始终零误差
- 政策变化前后历史报告自动保持最新

## 4. 退款与售后

- 已支付订单可根据服务进度进入人工审核退款流程
- 是否退款、退款比例和处理时长，取决于实际服务交付状态与争议情况
- 退款期间，相关支付/订单数据可能因审计需要暂不删除

## 5. 数据与隐私

- 具体隐私说明见 `docs/PRIVACY_POLICY_DRAFT.md`
- 数据保留与删除规则见 `docs/DATA_RETENTION_AND_DELETION.md`

## 6. 免责声明

- 志愿填报最终应以官方政策、招生章程、招生计划和考试院公布信息为准
- 本服务输出仅供参考，不替代学校、考试院或专业顾问的最终权威解释

## 7. 草案状态

本文件为产品上线前草案。  
在完成法务审阅、前台可见入口接入、版本化引用之前，不得对外宣称“正式服务协议已完成上线”。
