# 🚀 高考志愿填报 - 快速收集系统

## 3分钟快速版 vs 1分钟极速版

为不同场景设计了两套问卷：
- **3分钟版**：核心信息 + 详细类型匹配 + 约束条件
- **1分钟版**：仅保留最关键的5个问题

---

## ⚡ 3分钟快速问卷

### 使用场景
- 正常咨询，希望快速但完整收集
- 适合大部分考生

### 问卷内容

复制以下内容发送：

```
╔═══════════════════════════════════════════════════════════════════╗
║              ⚡ 高考志愿填报 - 3分钟快速问卷                        ║
╚═══════════════════════════════════════════════════════════════════╝

【必填】
1. 考生姓名：___________
2. 所在省份：___________
3. 高考总分：_____分
4. 全省位次：_____名（查一分一段表）

【核心匹配 - 选1个】
5. 你是哪种类型？
   R. 🔧 喜欢动手术拆装修东西 → 计算机、电子、机械
   I. 🔬 喜欢研究探索解难题 → 数学、物理、医学
   A. 🎨 喜欢创意创作画画 → 设计、建筑、传媒
   S. 🤝 喜欢帮助人当老师 → 师范、医学、心理学
   E. 📊 喜欢领导组织活动 → 管理、经济、法学
   C. 📝 喜欢规范有序整理 → 会计、统计、财务

6. 最擅长哪2门？（如：物理、数学）
7. 明确不喜欢？（A背诵 B数学 C社交 D实验 E代码）

【选填 - 有就填】
8. 经济条件：①困难 ②一般 ③中等 ④较好
9. 毕业规划：①直接工作 ②读研 ③考公 ④不确定
10. 工作地点：①一线 ②新一线 ③本省 ④都能接受
```

### 解析示例

**用户回复：**
```
1. 李明
2. 浙江
3. 612
4. 15230
5. R
6. 物理、数学
7. C
8. ③
9. ①
10. ②
```

**自动解析结果：**
```
📋 快速信息汇总
==================================================
👤 李明 | 浙江
📊 高考：612分
📊 位次：15230名

🎯 兴趣类型：现实型 (R)
   推荐专业：计算机、电子、机械、自动化

💪 优势学科：物理、数学
❌ 应避免：社交应酬

📍 规划：直接工作 | 地域：新一线城市
==================================================
✅ 核心信息完整！可以开始推荐
```

## ⚡ 1分钟极速版

### 使用场景
- 时间紧急，快速启动
- 考生只愿意回答最少问题
- 先收集核心，后续再补充

### 问卷内容

复制以下内容发送：

```
╔═══════════════════════════════════════════════════════════════════╗
║              ⚡ 高考志愿 - 1分钟极速版                              ║
╚═══════════════════════════════════════════════════════════════════╝

只需要回答这5个问题：

1️⃣ 姓名+省份：（如：李明 浙江）
2️⃣ 分数+位次：（如：612分 15230名）
3️⃣ 选科：（如：物化地）
4️⃣ 类型：（选1个字母）
   R=动手型 I=研究型 A=艺术型 S=助人型 E=管理型 C=常规型
5️⃣ 不能接受的：（如：背诵、数学、社交）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
示例回复：李明 浙江 612分 15230名 物化地 R 不接受社交
```

### 超简格式

甚至可以这样：

```
请按格式回复：姓名 省份 分数 位次 类型(RIASEC选一个) 不能接受的

例：李明 浙江 612 15230 R 社交
```

## 💡 使用技巧

### 1. 开场话术

**标准版：**
```
您好！为了给您匹配最适合的志愿方案，
请花3分钟填写以下信息，推荐会更精准。
```

**极速版：**
```
请直接回复：姓名 省份 分数 位次 类型 不能接受的
例：李明 浙江 612 15230 R 社交
```

### 2. 字段识别规则

系统会自动识别多种格式：

**格式1 - 带编号：**
```
1. 李明
2. 浙江
3. 612
4. 15230
5. R
```

**格式2 - 紧凑型：**
```
李明 浙江 612分 15230名 物化地 R 不接受社交
```

**格式3 - 自然语言：**
```
我叫李明，浙江考生，考了612分，位次15230，
选了物理化学地理，属于动手型，不喜欢社交
```

### 3. 缺失信息追问

如果信息不完整，系统会提示：

```
⚠️ 还需补充：位次、兴趣类型

请补充：
- 全省位次（查一分一段表）：_____
- 兴趣类型（R/I/A/S/E/C选一个）：_____
```

- **首次咨询** → 用3分钟快速版，快速出结果
- **深度规划** → 切换到7步完整版
- **紧急情况** → 用1分钟极速版，后续再补充

## 🛠️ 技术集成

### 在对话中使用

```python
# 导入快速收集模块
from gaokao_quick_3min import parse_quick_response, generate_quick_summary

# 用户输入
user_input = """1. 李明
2. 浙江  
3. 612
4. 15230
5. R
6. 物理、数学"""

# 解析信息
info = parse_quick_response(user_input)

# 生成摘要
summary = generate_quick_summary(info)
print(summary)

# 生成建议
recommendation = generate_quick_recommendation(info)
print(recommendation)
```

### 命令行使用

```bash

"
```

## ✅ 质量检查

使用快速问卷后，检查是否收集了：

- [x] **必填**：姓名、省份、分数、位次
- [x] **核心**：霍兰德类型（R/I/A/S/E/C）
- [x] **辅助**：优势学科、不喜欢什么
- [ ] **选填**：经济条件、毕业规划、地域偏好

如果选填项缺失，可以在推荐时询问：
```
还需要了解：
- 家庭经济情况（影响学费承受力）
- 毕业后的打算（就业/考研/考公）
```
