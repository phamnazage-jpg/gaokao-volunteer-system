## 1. 推广场景总览

### 1.1 渠道矩阵

```
                     线上                    线下
                    ┌───┐                  ┌───┐
         付费流量   │ 闲鱼│                  │   │
                    │  │  │                  │ 学校│
         内容种草   ├───┼───┤    口碑传播    │   │
                    │小红书│◄────────────────│师生│
         知识营销   │  │  │                  │   │
                    │ 知乎│                  │   │
                    └───┘                  └───┘
```

### 1.2 渠道定位

| 渠道       | 定位     | 场景                       | 交易地点         | 服务方式                 |
| ---------- | -------- | -------------------------- | ---------------- | ------------------------ |
| **闲鱼**   | 交易平台 | 用户看到商品→下单→提供服务 | 闲鱼内交易       | 闲鱼发方案 或 引流至微信 |
| **小红书** | 内容种草 | 发布案例→引流→成交         | 引导至闲鱼或微信 | 微信交付                 |
| **知乎**   | 知识营销 | 回答专业问题→引流→成交     | 引导至闲鱼或微信 | 微信交付                 |
| **学校**   | 口碑传播 | 学生/老师推荐→加微信       | 微信/企业微信    | 微信交付                 |
| **Web端**  | 自助服务 | 用户自主访问→支付→获取     | Web内支付        | 系统自动生成报告         |

## 7. 风险与合规

### 7.1 风险点

| 风险         | 影响         | 应对措施                      |
| ------------ | ------------ | ----------------------------- |
| 闲鱼违规下架 | 销售渠道中断 | 多备几个账号，不要留微信/电话 |
| 微信封号     | 私域流失     | 企业微信为主，个人微信备用    |
| 用户投诉     | 退款/差评    | 明确服务边界，免责声明清晰    |
| 退款纠纷     | 财务损失     | 闲鱼7天无理由，微信按进度退款 |

### 7.2 合规要求

- ✅ 闲鱼：不得留联系方式（可用图片/谐音）
- ✅ 微信：企业微信更佳，有企业背书
- ✅ 服务：明确"仅供参考，以官方为准"
- ✅ 退款：闲鱼按平台规则，微信协商退款
